# Code of Conduct

## Our Pledge

我们希望这个项目保持友好、开放、以学习为中心。所有参与者都应尊重不同背景、经验水平和观点的人。

## Expected Behavior

- 使用友善、包容、专业的表达。
- 聚焦问题、证据和改进方案。
- 尊重维护者的时间，提交清晰、可复现、范围明确的问题或 PR。
- 在讨论概率、赌博风险或教育内容时保持负责任的语气。

## Unacceptable Behavior

- 人身攻击、骚扰、歧视、威胁或持续打扰。
- 发布他人的私人信息，或要求他人公开敏感信息。
- 推广真实赌博、引流、套利承诺、诈骗或绕过法律限制的用途。
- 在 Issue、PR、讨论或示例中加入密钥、token、后台截图、个人账号或其他敏感内容。

## Enforcement

维护者可以编辑、隐藏或删除违反本准则的内容，也可以临时或永久限制相关参与者继续参与项目。处理问题时会尽量保持事实清楚、影响最小、对社区负责。

## Attribution

This Code of Conduct is adapted from common open source community standards and tailored for this project.
