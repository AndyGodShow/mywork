# Security Policy

## Supported Versions

当前只维护 `main` 分支。

## Reporting a Vulnerability

如果你发现安全问题，请优先通过 GitHub 私密漏洞报告功能联系维护者。如果该功能不可用，请创建一个不包含利用细节和敏感信息的 Issue，说明你希望私下报告安全问题。

请不要公开以下内容：

- 可直接利用的漏洞细节；
- API Key、token、cookie、私有部署链接；
- 个人身份信息、联系方式、钱包地址；
- 管理后台、云平台或部署平台截图。

## Project Scope

这个项目是前端教育模拟器，不处理真实支付、真实下注、账号资金、提现或 KYC 信息。任何引入这些能力的改动都不在当前项目目标内。
