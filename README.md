# 🎰 赌场教育模拟器 (Casino Education Simulator)

一个用于理解赌场游戏概率和数学原理的教育模拟器。通过可视化和批量模拟，帮助用户直观感受"庄家优势"和各种下注策略的长期效果。

> ⚠️ **本项目仅供教育用途，请勿用于真实赌博。**

## ✨ 功能列表

### 百家乐 (Baccarat)
- 8 副牌标准百家乐规则（含第三张牌规则）
- 多种下注方式：闲、庄、和、闲对、庄对
- 5 种下注策略的批量模拟对比
- 路单显示：珠盘路 + 大路
- 教育弹窗：概率数学原理讲解

### 二十一点 (Blackjack)
- 标准规则：Ace 双值、S17 庄家停牌
- 基本策略引擎：实时操作建议提示
- 3 种策略模拟：基本策略、倍投+基本策略、模仿庄家
- 资金曲线可视化

### 轮盘 (Roulette)
- 欧洲轮盘（37 格，含单 0）
- 多种下注：直注、红黑、单双、高低、打、列、街注、线注
- 3 种策略模拟：平注押红、倍投押红、随机外围注
- 资金曲线可视化

## 📖 教育意义

所有赌场游戏的期望值均为负数——这是由赌场规则在数学上保证的。通过本模拟器，你可以：

1. **验证概率论**：运行上万局模拟，观察实际胜率趋近理论值
2. **理解策略陷阱**：亲眼看到倍投法等"必胜系统"如何在长期失效
3. **感受波动**：短期可能赢钱，但长期必然回归数学期望

| 游戏 | 赌场优势 |
|------|---------|
| 百家乐（押庄） | 1.06% |
| 百家乐（押闲） | 1.24% |
| 二十一点（基本策略） | ~0.5% |
| 轮盘（欧洲） | 2.7% |
